### Subtracting Numbers (Operator -)

Substracting numbers is done using the **`-`** operator:

```csharp
var a = int.Parse(Console.ReadLine());
var b = int.Parse(Console.ReadLine());
var result = a - b;
Console.WriteLine(result);
```
