### Problem: excellent grade

The first exercise for this topic is to write a **console program** that **inputs the grade** (decimal number) and prints **Excellent!** if the grade is **5.50** or high.

#### Sample Input and Output

| Input | Output |
| --- | ---- |
| 6 | Excellent! |
| 5 | (no output) |
| 5.5 | Excellent! |
| 5.49 | (no output) |
