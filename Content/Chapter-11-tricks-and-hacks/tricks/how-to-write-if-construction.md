### How to Write a Conditional Construction?

The conditional **`if` construction** contains the following elements:

* Keyword **`if`**
* **A boolean expression** (condition)
* **Body** of the conditional construction
* Optional: **`else` clause**

```csharp
if (condition)
{
    // body
}
else (condition)
{
    // body
}
```
